# Contributing to CoBWeaverClaw

Thank you for your interest in contributing! 🕷️

## How to Contribute

### 1. Report Bugs
Open an issue with:
- Platform (Android/iOS/Linux/Windows/macOS)
- Steps to reproduce
- Expected vs actual behavior

### 2. Suggest Features
Open an issue labeled `enhancement` with:
- Problem description
- Proposed solution
- Why it's better than current behavior

### 3. Submit Code

```bash
# Fork the repo, then:
git clone https://github.com/YOUR_USERNAME/CoBWeaverClaw
cd CoBWeaverClaw
git checkout -b feature/your-feature-name

# Make changes, then:
git commit -m "feat: describe your change"
git push origin feature/your-feature-name
# Open a Pull Request
```

### 4. Add Skills
Skills are single Python files in `skills/`. See `skills/sandbox.py` for the base class.

### 5. Add Translations
Add your language file in `i18n/` following the pattern in `i18n/en.json`.

## Code Style
- Python 3.11+
- Type hints required
- Docstrings for all public methods
- Tests for new features

## Community
- Respect all contributors
- Arabic and English are both official languages
- Be constructive in reviews
